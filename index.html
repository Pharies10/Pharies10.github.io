Threading using C

Language used: C

A process of using multithreading and the monte carlo method in order to determine pi by throwing generating random point insided a 2x2 square and determining whether those points are inside a circle inside the square. There are two different files (pi1, pi2) that use different ways of recording data within the threading process.

Vertical Sort

Language used: Pyhton

A sorting that sorts a 2D array that sorts columns 4 seperate times. Each of the seperate sorts was used to determine what combinations of sorting algorithms (ex: quicksort, mergesort, bubblesort) would be the fastest with varying sizes of 2D arrays. The full report is linked to the Readme.
